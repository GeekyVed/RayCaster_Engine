# RayCaster Engine

RayCaster Engine is a powerful 3D environment rendering engine that utilizes ray casting techniques, developed with OpenGL, FreeGLUT, and C++. Experience immersive worlds and dynamic scenes with ease.

![RayCaster Demo](images/raycaster_demo.png)

## Project Overview

RayCaster Engine empowers you to create and explore 3D environments through the following features:

- **Ray Casting Engine**: Utilize our custom-built ray casting engine to render 3D perspectives within a 2D space, providing a captivating visual experience.
  
- **Map Creation**: Craft intricate maps with walls and empty spaces, laying the foundation for your virtual world.

![Map Creation](images/map_creation.png)

- **Efficient Ray Casting Algorithm**: Benefit from our optimized ray casting algorithm, ensuring accurate wall collision detection and realistic environment rendering.

- **Dynamic 3D Scene Rendering**: Immerse yourself in fully rendered 3D scenes with dynamic lighting effects and adaptable wall layouts, enhancing realism and immersion.
